Analisis Penyakit Jantung dengan Algoritma K-Nearest Neighbors (KNN)

Oleh Partogi Sitinjak | 25 Februari 2026

Proyek ini berfokus pada klasifikasi risiko penyakit jantung menggunakan dataset Heart_Disease.csv. Algoritma yang digunakan adalah K-Nearest Neighbors (KNN), sebuah metode yang mengklasifikasikan data berdasarkan jarak terdekat dengan tetangganya.

Langkah-Langkah Implementasi:

  • Data Cleaning: Mengolah data yang hilang (missing values) menggunakan Python di Jupyter Notebook.
  • Feature Selection: Memilih variabel penting seperti usia, tekanan darah, dan kadar kolesterol.
  • Model Training: Menggunakan Scikit-Learn untuk melatih model dengan membagi data menjadi set pelatihan dan pengujian.

Potongan Kode Python:

from sklearn.neighbors import KNeighborsClassifier
# Inisialisasi model dengan k=5
knn = KNeighborsClassifier(n_neighbors=5)
knn.fit(X_train, y_train)
print("Akurasi Model:", knn.score(X_test, y_test))

Hasil pengujian menunjukkan akurasi yang cukup baik. Proyek ini membuktikan bahwa pengolahan data yang tepat dapat membantu dalam prediksi awal kondisi kesehatan seseorang melalui teknologi Data Mining.
